# CLAUDE.md 初期化

プロジェクトのルートに CLAUDE.md ファイルを作成してください。

## 分析対象

以下を調査してコードベースを理解してください：

1. プロジェクト構造を確認
2. package.json, Cargo.toml, go.mod, requirements.txt などの依存関係ファイルを確認
3. README.md があれば参照
4. 主要なソースコードディレクトリを確認

## CLAUDE.md に含める内容

1. **プロジェクト概要**
   - 何をするプロジェクトか
   - 使用している技術スタック

2. **ディレクトリ構成**
   - 主要ディレクトリの役割

3. **開発コマンド**
   - ビルド方法
   - テスト実行方法
   - 開発サーバー起動方法

4. **重要なファイル**
   - 設定ファイル
   - エントリーポイント

5. **注意事項**
   - 開発時の注意点やルール

## 出力

プロジェクトルートに `CLAUDE.md` を作成してください。簡潔で実用的な内容にしてください。
