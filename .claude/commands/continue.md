# /continue

CLAUDE.md の引き継ぎセクションから作業を続行する。

## 手順

1. CLAUDE.md を読み、「引き継ぎ」セクションを確認
2. 未解決の問題と次のステップを把握
3. plan.md があれば残タスクも確認
4. 現状をユーザーに報告
5. どこから始めるか確認して作業開始

## 報告内容

- 前回完了した作業
- 未解決の問題（あれば）
- 次のステップ候補

## 確認するファイル

- `CLAUDE.md` - 引き継ぎセクション
- `plan.md` - チェックリスト（存在する場合）
- `HANDOVER.md` - 詳細な引き継ぎ書（存在する場合）
