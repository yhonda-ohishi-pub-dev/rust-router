# /handover

CLAUDE.md に引き継ぎ情報を追記し、既存の内容を整理する。

## 手順

1. このセッションで行った作業を振り返り、まとめる
2. 未解決の問題を特定し、原因と状況を記録
3. 次のステップをチェックリスト形式で記載
4. CLAUDE.md の末尾に「引き継ぎ」セクションを追加または更新
5. 古い引き継ぎセクションがあれば削除
6. 重複・陳腐化した情報を整理

## フォーマット

```markdown
## 引き継ぎ（YYYY-MM-DD）

### 完了した作業
- 作業内容

### 未解決の問題
- 問題: 原因と状況

### 次のステップ
- [ ] タスク
```
