# Plan Executor

## 概要

`plan.md` のチェックリスト項目を読み取り、順番に実行する。

## 実行手順

1. `plan.md` を読み取る
2. チェックリスト（`- [ ]` 項目）を抽出する
3. 未完了タスクを TodoWrite で管理する
4. 各タスクを順番に実行する
5. 完了したタスクは `- [x]` に更新する

## チェックリスト形式

```markdown
- [ ] 未完了タスク
- [x] 完了タスク
```

## 実行ルール

- 各フェーズを順番に進める
- タスク完了後は plan.md のチェックボックスを `- [x]` に更新
- エラーが発生したら報告して次のステップを相談
- 1つのタスクが完了したら次に進む前に確認
