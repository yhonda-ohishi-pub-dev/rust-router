# Continue Skill

CLAUDE.md の引き継ぎセクションから続行する。

## 実行手順

1. CLAUDE.md を読んで「引き継ぎ」セクションを確認
2. 未解決の問題と次のステップを把握
3. plan.md があれば残タスクも確認
4. 状況をユーザーに報告し、何から始めるか確認

## 報告フォーマット

```markdown
## 引き継ぎ状況（YYYY-MM-DD）

### 前回の作業
- [完了した作業の要約]

### 未解決の問題
- [問題]: [状況]

### 次のステップ
1. [ ] タスク1
2. [ ] タスク2

---

どこから始めますか？
```

## 注意事項

- 引き継ぎセクションがない場合は、CLAUDE.md 全体から現状を把握
- plan.md のチェックリストと照合して進捗を確認
- ユーザーの指示を待ってから作業開始
