# Handover Skill

引き継ぎ情報を CLAUDE.md に追記し、既存の内容を整理する。

## 実行手順

1. 現在のセッションで行った作業を振り返る
2. 未解決の問題・次のステップを特定する
3. CLAUDE.md の「引き継ぎ」セクションを更新する
4. 古くなった情報や重複を整理・削除する

## 引き継ぎフォーマット

```markdown
## 引き継ぎ（YYYY-MM-DD）

### 完了した作業
- [作業1]
- [作業2]

### 未解決の問題
- [問題1]: [原因と状況]
- [問題2]: [原因と状況]

### 次のステップ
- [ ] [タスク1]
- [ ] [タスク2]
```

## 整理のガイドライン

- 古い引き継ぎセクションは削除（最新のみ保持）
- 重複した情報は統合
- 完了済みのタスクは「完了した作業」に移動
- 行番号参照（`file.rs:123`）はコード変更で陳腐化するため削除
